Easy Jogging

Easy Jogging has been made with a problem in mind:
Where should I jog if I want to jog a determined number of kilometers?

The app generates a random path from your position, allowing you to run in various environment every time you want to jog. A stopwatch and an history of your paths are also featured in order to manage your previous generated path and time.

Easy Jogging is a project application made using intel XDK, based on Cordova. I used HTML, CSS and Javascript to develop it as well as the Google map API. The app runs on Android and Iphone.

I have some experience in Android development and working with web technologies to create an app was very useful. The Intel XDK IDE was also very helpful to emulate, test and generate apks in the same environment.
